Frontend (React + Next.js + Tailwind)
Node.js: >=18.x
React 18 and Next.js 13+ require Node 18 or higher for full features (App Router, Server Components, etc.).
TailwindCSS works well with Node 18+.
npm: >=9.x (comes with Node 18)

Backend (FastAPI + Python + Snowflake)
Python: 3.9 – 3.11
FastAPI supports 3.7+, but Snowflake connector and modern libraries work best on Python 3.9+.
Python 3.12 is new and may have minor compatibility issues with Snowflake connector.
pip: Latest version (upgrade if needed)
-------------------------------------------------------
Reach Front end commands:
npx create-next-app@latest <AppName>
npm install react-icons
npm install recharts react-icons
npm run dev
npm install react-leaflet leaflet
npm install recharts
Install the types package:
npm install --save-dev @types/leaflet


Backend:

Create Venv
python -m venv .venv

Activate venv 
.venv\Scripts\activate

Install required packages:
pip install fastapi uvicorn snowflake-connector-python pandas python-dotenv
pip install snowflake-connector-python
Environment Variables
Create .env in the root folder:


│── app/
│   ├── __init__.py
│   ├── main.py              # Entry point
│   ├── config.py            # Configuration & credentials
│   ├── database.py          # DB connection logic
│   ├── routers/
│   │   ├── __init__.py
│   │   ├── historical.py    # API for historical data
│   │   └── forecast.py      # API for forecast data
│   └── services/
│       ├── __init__.py
│       ├── historical_service.py
│       └── forecast_service.py
│── .env                     # Environment variables
│── requirements.txt
│── README.md
